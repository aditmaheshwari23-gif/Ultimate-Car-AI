# Car AI Assistant (Jupyter Notebook Project)

## Overview

This project is an interactive Car Data Analysis and AI-style assistant built using Python and Pandas inside a Jupyter Notebook. It allows users to explore, filter, compare, and analyze car datasets using natural language-style input.

The system supports smart querying such as brand search, price filtering, fuel type filtering, and car comparisons.

---

## Features

- Search cars by brand name
- Filter cars by price range
- Find cheapest and most expensive cars
- Filter by fuel type (diesel, petrol)
- Compare two car brands side-by-side
- Intelligent natural language-style input handling
- Instant dataset insights (average price, total results)
